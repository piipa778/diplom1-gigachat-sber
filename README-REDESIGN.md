# Что изменено в редизайне

Стек проекта сохранён: Node.js + Express + PostgreSQL для отзывов, статический фронт без перехода на другой фреймворк.

## Основные улучшения

- Полностью переработаны адаптивные стили в `css/site-custom.css`.
- Добавлен современный sticky-header с blur-эффектом.
- Добавлено мобильное меню без сторонних библиотек в `js/site-custom.js`.
- Улучшены карточки меню, услуг, отзывов, контактов и новостей: тени, скругления, hover-анимации.
- Улучшены формы, кнопки, CTA-блоки и типографика.
- Добавлены mobile-first правки для экранов до 1050px, 760px и 520px.
- Все 31 HTML-страницы уже подключают `site-custom.css`, `site-custom.js` и viewport для адаптивности.

## Запуск

1. Создайте `.env` по примеру `.env.example` и укажите `DATABASE_URL`.
2. Установите зависимости:

```bash
npm install
```

3. Запустите проект:

```bash
npm start
```

4. Откройте сайт:

```text
http://localhost:3000
```

## Проверка

Проверены синтаксис `server.js` и `js/site-custom.js`. Backend-логика отзывов не менялась.
